# Periplus Shopping Cart Automation

This project automates Scenario B: testing shopping cart functionality on https://www.periplus.com/.

## Technology

- Java
- Selenium WebDriver
- TestNG
- Maven
- WebDriverManager

## Automated Scenario

1. Open Chrome
2. Navigate to Periplus
3. Login with a registered test account
4. Search for one product
5. Add product to cart
6. Verify product is added to cart

## Environment Variables

Before running the test, set:

```bash
PERIPLUS_EMAIL=your-email@example.com
PERIPLUS_PASSWORD=your-password
```
